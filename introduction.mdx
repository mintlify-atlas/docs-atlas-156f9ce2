# Windows-MCP

**Windows-MCP** is a lightweight, open-source MCP server that enables seamless integration between AI agents and the Windows operating system. Acting as a bridge between LLMs and Windows, it allows agents to perform tasks such as **file navigation, application control, UI interaction, QA testing**, and more.

## Key Features

<CardGroup cols={2}>
  <Card title="Native Windows Integration" icon="windows">
    Interacts natively with Windows UI elements via UIAutomation, opens apps, controls windows, simulates user input, and more
  </Card>
  <Card title="Works with Any LLM" icon="brain">
    No computer vision required - works with any LLM, reducing complexity and setup time
  </Card>
  <Card title="16 Powerful Tools" icon="toolbox">
    Comprehensive toolset including App, Shell, FileSystem, Snapshot, Click, Type, and more
  </Card>
  <Card title="Local & Remote Modes" icon="server">
    Run locally on your machine or connect to cloud-hosted Windows VMs
  </Card>
  <Card title="Real-Time Interaction" icon="bolt">
    Typical latency of 0.2-0.9 seconds between actions
  </Card>
  <Card title="DOM Mode for Browsers" icon="globe">
    Special mode for efficient web automation that focuses on page content
  </Card>
  <Card title="Multi-Monitor Support" icon="display">
    Full support for multi-monitor setups and virtual desktops
  </Card>
  <Card title="Lightweight & Open Source" icon="code">
    Minimal dependencies with full MIT-licensed source code
  </Card>
</CardGroup>

## Supported Operating Systems

Windows-MCP works on:

- Windows 7
- Windows 8, 8.1
- Windows 10
- Windows 11
