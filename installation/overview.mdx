<Note>
When you install Windows-MCP for the first time, it may take a minute or two to install dependencies. In the first run, the server may timeout - ignore it and restart.
</Note>

## Prerequisites

Before installing Windows-MCP, ensure you have:

- **Python 3.13+** installed on your Windows system
- **UV (Package Manager)** from Astral
- **English** as the default language in Windows (preferred, or disable the `App-Tool` for other languages)

### Installing UV

You can install UV using either of these methods:

<CodeGroup>

```bash pip
pip install uv
```

```bash curl
curl -LsSf https://astral.sh/uv/install.sh | sh
```

</CodeGroup>

## Supported Operating Systems

Windows-MCP works on all modern Windows versions:

- Windows 7
- Windows 8, 8.1
- Windows 10
- Windows 11

### PyPI Installation

The easiest way to install Windows-MCP is using `uvx` to run the latest version directly from PyPI:

```json
{
  "mcpServers": {
    "windows-mcp": {
      "command": "uvx",
      "args": [
        "windows-mcp"
      ]
    }
  }
}
```

### Source Installation

For development or customization, you can install from source:

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/CursorTouch/Windows-MCP.git
    cd Windows-MCP
    ```
  </Step>
  <Step title="Configure your MCP client">
    ```json
    {
      "mcpServers": {
        "windows-mcp": {
          "command": "uv",
          "args": [
            "--directory",
            "<path to the windows-mcp directory>",
            "run",
            "windows-mcp"
          ]
        }
      }
    }
    ```
  </Step>
</Steps>
